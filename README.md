# Demo of NAOC Python Training

本项目是2019年国家天文台 [“天文数据与Python技术培训”](https://hebl.china-vo.org/course/PIA2019/) 的案例。

## 克隆与安装

克隆本项目

```bash
git clone https://github.com/wangleon/NAOCTraining.git
```

进入项目所在的目录

```bash
cd NAOCTraining
```

安装

```bash
sudo python3 setup.py install
```

## 编译项目文档

进入文档所在的目录

```bash
cd doc
```

编译html格式的文档
```bash
make html
```
